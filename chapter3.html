Strings

Q1.Write a python program to display a user entered name followed by Good Afternoon using input() function

name = input("Enter your name")
print("Good Afternoon "+name)

Q2.Write a Python program to fill in a letter template given with name and date.

import datetime as d
name = input("Enter Name")
dateToday = d.date.today()

print(f"Dear {name},\nYou are selected!\n {dateToday}")

Q3.Write a Python program to detect double spaces in a string

text = "This is  the string  having double spacess  "

print(text.count("  "))
#count() function count the number of occurence (how many times)

Q4.Replace the double spaces from problem 3 with single spaces.

text = "This is  the string  having double spacess  "
text_single_space = text.replace("  "," ") #repalce("d-space", "s-space")
print(text_single_space)
