Q1.Write a program to read the text from a given file "poem.txt", and find out wether it contains the word "twinkle".

with open("poems.txt","r") as f:
	data = f.read()
	data = data.lower()
	if "twinkle" in data:
		print("Yes Twinkle is present")
	else:
		print("No Twinkle is not present")

Q4.A file contains a word "Donkey" multiple times. You need to write a program which replace this word with ##### by updating the same file.

	with open("donkey.txt", "w") as f:
		f.write(text)
